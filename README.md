First try to build a client-server project with :
- Backend = Scala + Play framework
- Frontend = Scala.js + Laminar + Laminext (websocket)

**Usage** : 
- Back
  - open a new console
  - cd AgilePokerBackEndWeb
  - sbt run
- Front
  - open a new console
  - cd AgilePokerFrontEnd
  - launch sbt
  - fastOptJS
